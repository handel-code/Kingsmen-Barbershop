# Kingsmen Barbershop 

A fully responsive multi-page web application for a premium barbershop in Westlands, Nairobi.

## Pages
- **Home** - Hero section, highlights, why choose us, featured services
- **About** - Shop story, values, and meet the team
- **Services** - Full pricing list with 9 services
- **Gallery** - Filterable photo grid with 4 categories
- **Booking** - Appointment form with validation and info sidebar
- **Reviews** - Client reviews with rating summary
- **FAQ** - Accordion Q&A with live search
- **Contact** - Contact info and message form

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5.3

## JavaScript Features
1. Gallery filter - filter cuts by category
2. Booking form validation - with date checking
3. Contact form validation
4. Dark/Light mode toggle
5. Navbar scroll effect
6. Scroll fade-in animations
7. FAQ live search
